# Job_Parser (Парсер вакансий)
Получает информацию о вакансиях с разных платформ в России, сохраняет ее в файл и позволяет удобно работать с ней (добавлять, фильтровать, удалять)

# Выходные данные

- Информация о вакансиях, полученная с разных платформ, сохраненная в JSON-файл.
- Отфильтрованные и отсортированные вакансии, выводимые пользователю через консоль.
